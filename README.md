# Chess-Bet

A realtime multiplayer betting platform supporting Chess and Xiangqi.

## Tech Stack
- Frontend: Next.js 14, Tailwind CSS, Zustand, Socket.IO Client
- Backend: Fastify, Socket.IO, Prisma ORM, TypeScript
- Database: SQLite (PostgreSQL compatible)

## Features
- Real-time multiplayer board sync
- 2 game engines (Chess.js, React-Xiangqiboard)
- Waitlist and Lobby mechanics
- Spectator Mode
